Madrid Urban Heat Island Effect (UHI) Project


This project aims to showcase a GitHub-hosted webmap that explores the Urban Heat effect in Madrid during a period of heatwave, by using statistical and GIS methods to find relationships for vegetation, spatial configuration, and temperature of Madrid.

One of the four objectives from this project was to: Examine relationships for vegetation and temperature to produce models and a GitHub-hosted webmap.

For the Class values (LAND SURFACE TEMPRETURE) to understand the tempreture readings:
Class 1 = 32-36 (°C)
Class 2 = 36-40 (°C)
Class 3 = 40-44 (°C)
Class 4 = 44-48 (°C)
Class 5 = 48-52 (°C)

For the the NDVI values to understand the vegetation readings: 
NDVI 1 = -1-0
NDVI 2 = 0-0.3
NDVI 3 = 0.3-0.5
NDVI 4 = 0.5-0.8
NDVI 5 = 0.8-1
